Дефиниравме 3 инфраструктури: клиент, веб-сервер и апликациски сервер.
Во клиентската инфраструктура, веб-прелистувачот испраќа повик API до презентацискиот слој кој се наоѓа во инфраструктурата на веб-серверот каде се сместени и сервисниот слој и слојот за интеграција.
Компонентата Data Source Access од интеграцискиот слој испраќа повик за читање/запишување назад до компонентата DataBase која се наоѓа во инфраструктурата на Application Server.
Service-Module,која што исто така се наоѓа во интеграцискиот слој, испраќа повик назад до Сервисниот модул лоциран во Operation systems processes and context, а компонентата Anti-police испраќа повик API за додавање податоци на компонентата DataBase.
